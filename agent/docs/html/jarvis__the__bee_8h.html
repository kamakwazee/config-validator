double compare_configs ( ConfigSet  baseline_set,
ConfigSet  compare_set 
)

Compares two sets of strings using the Jaccard similarity.

\[ J = \frac{M_{11}}{M_{10} + M_{01} + M_{11}}, \]

where \( M_{11} \) is the number of elements in both sets, \( M_{10} \) is the number of elements in the first set only, and \( M_{01} \) is the number of elements in the second set only.

Parameters
baseline_setThe baseline comparison
compare_setThe set of strings being compared
Returns
The Jaccard similarity between the sets of strings
